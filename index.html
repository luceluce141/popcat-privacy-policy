PopCat Clicker is designed to be a safe, casual game. The app itself does not collect, store, or process any personally identifiable information.

üì±2. App permissions and local game data

PopCat Clicker uses only the minimal permissions required to run the game and display ads. We do not request dangerous permissions such as access to contacts, SMS, or device storage for personal files.

Local game data

Game progress (for example collected items, coins, and upgrades) is stored locally on your device in standard game save files or preferences. This information:

  • Is not personal data and cannot identify you as an individual
  • Is never uploaded to our own servers
  • Is removed if you uninstall the game or clear its data
Network usage

The game may access the internet only to:

  • Load and display third-party advertisements
  • Check for updated ad content from providers

The game does not implement its own analytics or tracking SDKs, and we do not operate any backend server for this title.
